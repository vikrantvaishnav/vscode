HTML Presentational and Formatting Tags: An In-Depth Look

This page provides an extensive demonstration of various HTML tags specifically designed for text formatting and presentation. It highlights how these tags influence the visual appearance and semantic meaning of content without relying on external CSS.

Text Styling and Emphasis Tags

These tags are used to apply common text styles like bold, italic, underline, and to convey different levels of emphasis or semantic meaning.

Bold and Strong Emphasis

This sentence contains text that is rendered in a bold font style using the b tag. The b tag is typically used for visually bolding text without adding any special semantic importance.
Conversely, this sentence has text that conveys strong importance using the strong tag. While it often renders as bold, its primary purpose is to indicate that the content in of greater importance or seriousness.
Consider using strong for keywords in a warning or critical information, and b for names in an article or product names that don't need semantic emphasis

Italic and Emphasized Text

Here, we have text that is displayed in an italic font using the i tag. The i tag is generally used for text that needs to stand out vimially, such as technical terms, foreign words, or thoughts.
This sentence showcases text that is emphasized using the em tag. The em tag adds semantic emphasis, implying that the content should be stressed or pronounced differently.
For instance, une em when you want to emphasize a word in a sentence to change its meaning, and i for titles of works or scientific names

Underlined and Inserted Text

This text is underlined using the u tag. Historically, this was used for visual underlining, but it's now often discouraged for non-hyperlink text to avoid confusion. It can be used to represent text that is unarticulated but explicitly rendered as if underlined, such as misspellings.
This text is inserted using the ins tag. This tag semantically indicates content that has been added to a document, often rendered with an underline You can also specify a datetime attribute for ins to indicate when the content was inserted: New content added on Jan 15.2023.

Subscript and Superscript

Chemical formulas often use subscripts, like H2O (water) where the 2' is a sub elemenit.
Mathematical expressions or ordinal indicators use superscripts, such as E=mc2 or 10th, where the '2' and 'th' are sup elements.

Marked (Highlighted) Text

The mark tag is used to highlight or mark text that is of particular relevance in the context of the surrounding content, often rendered with a yellow background.

Small Text

This is normal text, and this is small text using the small tag. It typically represents side comments, copyright, or legal disclaimers.

Cited Work

The cite tag is used to denote the title of a work. For example, as famously quoted from Hamlet by William Shakespeare, "To be or not to be, that is the question."
Another example: The theory was first proposed in On the Origin of Species

Code, Variables, and Preformatted Text

These tags are crucial for displaying programming code, user input, sample output, and preserving text formatting

Code Snippets

For inline code, use the code tag. For example, to print "Hello, World!" in Python, you would use print("Hello, World!").
When defining a function in JavaScript: function greet() { console.log("Hi!"); }

Keyboard Input

The kbd tag indicates user input, typically from a keyboard. To save a document, press Ctrl + s.
To open the task manager on Windows, press Ctrl + Shift + Esc.

Sample Output

The samp tag represents sample output from a computer program. If the command fails, you might see: Error: Permission denied. After a successful compilation, the console might display: Build successful (0 warnings, 0 errors).

Quotes and Blockquotes

These tags are used for quoting text from other sources, distinguishing between short inline quotes and longer, block-level quotes

Short Quotes

The q tag is used for short, inline quotations. My friend said, I'll be there in five minutes! before hanging up.
Another common phrase is, Actions speak louder than words

Blockquotes

The blockquote tag is used for longer quotations that are typically displayed as a block. It often includes a citation. "Two roads diverged in a yellow wood, And sorry I could not travel both And be one traveler, long I stood And looked down one as far as I could To where it bent in the undergrowth;" -Robert Frost, The Road Not Taken This blockquote demonstrates how a larger section of quoted text is rendered, typically with indentation, and can include a citation using the cite tag.
